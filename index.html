iOS Instructions:

  1. Tap the "Download Contact vCard" link above.
  2. Tap "Download" in the prompt.
  3. Open the downloaded file in the Files app.
  4. Tap the vCard file, then tap "Add to Contacts".

Android Instructions:

  1. Tap the "Download Contact vCard" link above.
  2. The file should download automatically.
  3. Open the downloaded file (usually in the "Downloads" folder).
  4. Select the Contacts app to import the contact.

General Instructions:

  1. Tap the "Download Contact vCard" link above.
  2. Locate the downloaded vCard file on your device.
  3. Open the file to add the contact information to your address book.
